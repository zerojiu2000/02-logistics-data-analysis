# 物流配送与销售数据分析

## 项目简介
基于某企业6种商品的送货及用户反馈数据，进行多维度的物流配送服务分析、销售区域潜力挖掘和商品质量评估。

## 核心分析
- 配送服务问题诊断：识别配送时效较差的线路（货品4→西北，货品2→马来西亚）
- 销售潜力区域挖掘：发现货品2在华东地区有较大市场空间，西北地区需减少投入
- 商品质量问题评估：货品1、2、4存在质量问题，需加强质检

## 技术栈
- Python 3.7+
- Pandas (数据处理)
- NumPy (数值计算)
- Matplotlib (数据可视化)
- Jupyter Notebook
