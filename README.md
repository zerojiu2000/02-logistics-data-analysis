# 物流配送与销售数据分析项目

基于某企业6种商品的送货及用户反馈数据，进行多维度的物流配送服务分析、销售区域潜力挖掘和商品质量评估。

## 项目亮点

- 系统性分析：涵盖数据清洗、规整、分析和可视化全流程
- 业务导向：针对配送时效、销售潜力、产品质量三大业务问题
- 多维视角：时间维度、空间维度、产品维度多角度分析
- 实用结论：提供可落地的业务优化建议

## 分析目标与结论

### 1. 配送服务问题诊断
核心问题：识别配送时效较差的线路

主要发现：
- 严重问题线路：
  - 货品4到西北地区：按时交货率仅24.14%
  - 货品2到马来西亚：按时交货率仅10.00%
- 区域问题：西北地区整体按时交货率最低（27.87%）
- 时间趋势：第四季度（特别是11月）配送时效下降

建议：优先优化货品4到西北、货品2到马来西亚的物流线路

### 2. 销售潜力区域挖掘
分析方法：结合销售金额、数量、拒货率综合分析

潜力区域：
- 重点投入：货品2在华东地区（销售量大，质量合格率高）
- 谨慎投入：货品2在西北地区（拒货率高，配送时效差）
- 稳定发展：货品5在泰国、货品6在马来西亚表现良好

建议：加大货品2在华东地区的市场投入，优化西北地区策略

### 3. 商品质量问题评估
问题商品：
- 货品1：返修问题较突出
- 货品2：拒货率较高（尤其在西北地区）
- 货品4：质量问题与配送问题并存

建议：对货品1、2、4扩大抽检范围，加强质量控制流程
